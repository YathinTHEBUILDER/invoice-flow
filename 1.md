### 2. Replace institutional wording

InvoiceFlow is for individual investors, not institutional financiers.

Replace wording like:

- Providers
- Capital providers
- Financiers
- Institutional financiers
- Lending partners

With safer wording like:

- Investors
- Marketplace investors
- Verified investors
- Individual investors

Example:

```tsx
Pool: 14 Providers
````

Should become:

```tsx
Pool: 14 Investors
```

---

### 3. Make return/yield language legally safer

Do not present returns as guaranteed.

Avoid:

* Guaranteed returns
* Fixed returns
* Predictable ARR
* Earn 15%
* Assured yield
* Risk-free returns

Use:

* Target return
* Indicative return
* Estimated return
* Expected return range
* Returns vary by invoice risk, buyer profile, and tenure
* Investments carry repayment, delay, and default risk

Example change:

```tsx
45 Days / 13.5%
```

Should become:

```tsx
45 Days / Target Return
```

Add microcopy:

```tsx
Indicative return shown after invoice approval and risk review.
```

Investor-facing disclaimer:

```txt
InvoiceFlow is a marketplace for invoice-backed opportunities. Investments carry repayment, delay, and default risk. Returns are not guaranteed. Please review each invoice and risk disclosure before investing.
```

This disclaimer should appear on investor-related pages and marketplace previews.
